Some more software recommendations to go with my [previous post](/posts/2024-01-software/), in no particular order: 

[**Pureline**](https://github.com/chris-marsh/pureline): Beautify your terminal prompt, with Git and virtual environment information. You will need to install a font with Powerline glyphs -- [here](https://github.com/powerline/fonts) is a good list of such fonts.

[**ngrok**](https://ngrok.com/): Easily serve a simple website from your home computer with this free public reverse proxy service.

[**pipx**](https://pipx.pypa.io/): Install python apps in an isolated environment. I currently use this for [Poetry](https://python-poetry.org/docs/), so that I can install additional plugins not available in the Ubuntu repos (i.e. the "python3-") packages.

[**erd-editor**](https://docs.erd-editor.io/): Good for diagramming database schemas; web and VSCode extension versions exist.

[**wxMaxima**](https://wxmaxima-developers.github.io/wxmaxima/index.html) Computer algebra system with graphical interface and plots.

[**Whatsie**](https://snapcraft.io/whatsie): Whatsapp for Linux.

[**The Battle for Wesnoth**](https://www.wesnoth.org/): Completely free and open-source game, and a good one (RPG genre).

**Console games**

- 2048: https://packages.ubuntu.com/noble/2048
- nskake: https://packages.ubuntu.com/noble/nsnake
- vitetris: https://packages.ubuntu.com/noble/vitetris
