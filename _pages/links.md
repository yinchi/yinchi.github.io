Cool software for academics

* [LyX](https://www.lyx.org/) -- Cool document editor for academics.  Uses its own file format, but easily exports to LaTeX and PDF.
* [SumatraPDF](https://www.sumatrapdfreader.org/free-pdf-reader) -- Unlike Adobe, does not keep a file lock on open PDFs, making
  working with LaTeX/LyX a lot easier.
* [Notepad++](https://notepad-plus-plus.org/), an excellent text/code editor for Windows.
* [Inkscape](https://inkscape.org/), for vector drawings.
* [Jabref](https://www.jabref.org/), for reference management.
